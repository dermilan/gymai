# Gym Progress AI

MVP Flutter app for logging workouts and generating AI workout suggestions.

## Getting started

- Install the Flutter SDK and ensure the `flutter` command is available.
- Run `flutter pub get` to install dependencies.
- Run `flutter run` to launch on a device or emulator.

## Notes

- Configure the OpenRouter client in AppServices with your API key and model.
- Local storage is in-memory for now and will reset on app restart.
